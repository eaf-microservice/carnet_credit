# Carnet credit - كناش الكريدي 🛒💼

**Carnet Credit**

هو تطبيق متطور يهدف إلى رقمنة عملية تتبع الديون (الكريدي) التقليدية بين أصحاب المتاجر والزبائن. يحل التطبيق محل "الكناش" الورقي القديم بنظام سحابي آمن وسهل الاستخدام.

---

## 🚀 المميزات الرئيسية

### 🏪 لأصحاب المحلات (Shop Owners)

- **لوحة تحكم كاملة**: إدارة جميع الكليان وتتبع إجمالي الديون المستحقة.
- **ثنائية العملة (درهم/ريال)**: إمكانية التبديل بين العرض بالدرهم أو الريال المغربي لتسهيل التعامل اليومي بالحسابات الشعبية.
- **إدارة الهوية والأسماء**: إمكانية إضافة "لقب" للزبائن (مثل با لعروبي) لسهولة التعرف عليهم، مع الحفاظ على أسمائهم الحقيقية عند الربط.
- **شفافية تامة**: عرض تفاصيل العمليات لكل زبون بالوقت والتاريخ.
- **التحكم الإداري**: إمكانية تعديل الأخطاء في الأثمنة أو حذف العمليات الخاطئة مع تحديث الرصيد تلقائياً.
- **رقم هاتف دولي**: دعم أرقام الهواتف الدولية مع التحقق من صحتها لسهولة التواصل.
- **ربط فوري**: إنشاء رمز QR خاص بالمحل ليقوم الزبائن بمسحه والارتباط بالمحل فوراً.
- **دمج الحسابات**: إمكانية ربط الزبائن المضافين يدوياً بحساباتهم الحقيقية عند تسجيلهم في التطبيق.

### 👤 للزبائن (Customers)

- **تتبع محلات متعددة**: إمكانية التعامل مع عدة متاجر في نفس الوقت وعرض رصيد كل متجر على حدة.
- **تقييد ذاتي**: يمكن للزبون تسجيل مشترياته بنفسه (مع عدم إمكانية الحذف لضمان النزاهة).
- **آخر العمليات**: عرض تاريخ مفصل لآخر المشتريات والمدفوعات لكل محل.

---

## 🛠 التقنيات المستخدمة (Tech Stack)

- **Flutter**: إطار العمل لبناء تطبيق يعمل على جميع المنصات.
- **Firebase Auth**: لتأمين الحسابات ودعم **Google Sign-In** مع التحقق من البريد الإلكتروني (Gmail).
- **Cloud Firestore**: قاعدة بيانات سحابية لحظية (Real-time) لضمان تزامن البيانات بين البائع والمشتري.
- **Intl Phone Field**: لدعم أرقام الهواتف العالمية والمحلية بدقة.
- **GoRouter**: لإدارة التنقل داخل التطبيق بشكل سلس.
- **Provider**: لإدارة حالة التطبيق (State Management).

## 📦 كيفية التشغيل والتثبيت

1.  **المتطلبات**: تأكد من تثبيت Flutter SDK و Java.
2.  **تحميل المشروع**:
    ```bash
    git clone https://github.com/eaf-microservice/carnet.git
    cd carnet
    ```
3.  **تثبيت المكتبات**:
    ```bash
    flutter pub get
    ```
4.  **إعداد Firebase**:
    - قم بإنشاء مشروع جديد في [Firebase Console](https://console.firebase.google.com/).
    - قم بتشغيل الأمر التالي لربط المشروع:
      ```bash
      flutterfire configure
      ```
5.  **التشغيل**:
    ```bash
    flutter run
    ```

### 📄 Full Description

**كناش (Carnet)**
هو الحل الرقمي الأمثل لأصحاب المحلات والمتاجر في المغرب لتجاوز مشاكل "الكناش" الورقي التقليدي. صُمم التطبيق خصيصاً لتسهيل عملية تتبع الديون (الكريدي) وضمان حقوق التاجر والزبون في آن واحد.

#### 🌟 لماذا تختار تطبيق كناش؟

يجمع كناش بين البساطة في الاستخدام والقوة في الأداء، مع دعم كامل للخصوصية وتزامن البيانات اللحظي.

#### 🏪 مميزات لأصحاب المحلات (Shop Owners):

- **إدارة شاملة**: تتبع جميع ديون الزبائن في مكان واحد مع معرفة إجمالي المستحقات بضغطة زر.
- **تسجيل الخلاص**: إمكانية تسجيل المبالغ التي يدفعها الزبائن (تسديد الدين) وتحديث رصيدهم فوراً.
- **ثنائية العملة (درهم/ريال)**: ميزة حصرية تتيح لك التبديل بين الدرهم والريال المغربي لتسهيل الحسابات مع الزبائن حسب العرف المحلي.
- **هوية الزبون**: إمكانية إضافة ألقاب للزبائن لسهولة التعرف عليهم (مثلاً: "با محمد البقاش") مع الحفاظ على دقة البيانات.
- **شفافية كاملة**: كل عملية (شراء أو تسديد) مسجلة بالوقت والتاريخ، مع إمكانية تصحيح الأخطاء.
- **ربط فوري عبر QR**: أنشئ رمز QR لمتجرك ودع الزبائن يرتبطون بحسابك لمتابعة ديونهم بأنفسهم.

#### 👤 مميزات للزبائن (Customers):

- **تتبع محلات متعددة**: تابع ديونك في مختلف المتاجر (البقال، الجزار، إلخ) من تطبيق واحد.
- **سجل العمليات**: اطلع على تاريخ مشترياتك ومدفوعاتك لحظة بلحظة.
- **نزاهة البيانات**: نظام "الزبون يضيف والتاجر يراجع" يضمن عدم ضياع الحقوق أو التلاعب بالسجلات.

#### 🔐 أمان وخصوصية:

بياناتك محفوظة في سحابة آمنة ومرتبطة بحسابك في Google، مما يضمن وصولك إليها من أي جهاز وفي أي وقت دون خوف من ضياع الكناش الورقي أو تلفه.
